### 继承
- 子类可以使用父类定义的内容或者行为等
- 继承的实现:
    - 父类，基类，被继承的类，Base Class,Super Class
    - 子类，有继承行为的类
    - 所有类都必须有一个父类
        - 如果没有，则默认是object的子类
        - 子类可以有多个父类
        - ~~~~Python
        class  Person():
            pass
            
            # 父类写在类定义的时候的括号里
        class Teacher(Person):
            pass                                 
        ~~~~

### issueclass检测是否为子类
- 可以用来检测两个类的父子关系
    - 使用~~~~
    issueclass(类名，类名)
    ~~~~
### 构造函数的继承
- 构造函数默认继承
- 一旦子类定义了构造函数，则不再自动调用父类的构造函数
~~~~Python
class Person():
    def __init__(self):
        pass

class Teacher(Person):
    pass
~~~~        



- 一个类由功能和特征两个内容组成
    - 特征就是一个描述
        - 特征在编程中就是一个变量，在类中成为**属性**
    - 功能就是一个能力
        - 功能在编程中就是一个函数，在类中成为**方法**
        
    - 类中属性一般定义在前面，方法定义在后面

- 对象成员的操作
    - 在类的外部，使用对象操作成员
        - 通过对象访问(修改)类中的属性
            - 如果是添加，则别的实例不会有添加了的属性
            - 如果是修改，则别的实例仍然会有修改了的属性
            - 删除一个对象的属性时，只能删除当前这个对象自己的属性，而不可以删除原本类中拥有的属性
            - 删除一个对象的属性，只能删除这个对象自己的属性，包括给对象添加的和修改的
            - 对于给对象自己添加的或者修改的属性，就相当于给这个对象赋予了自己的属性，如果对其进行删除，那么还想调用的话，就会去原来的类中去寻找对应的属性
            
        - 通过对象访问类中的方法
            - 访问对象的方法:实际上如果这个对象没有自己的独立的方法，会访问这个对象的类的方法
            - 修改对象的方法:给这个对象的方法重新定义
            - 添加对象的方法:给这个对象自己新创建的方法(和修改极为类似)
    - 总结
        - 实际上，创建对象的时候，并不会把类的属性和方法复制一份给对象，而是在对象中引用父类的方法
        - 因此在访问对象的属性时，会先去找对象自己的属性，如果没有就去找这个类的属性和方法
        - 一个对象由类创建以后，就是一个独立的对象，会引用父类的属性和方法
        - 如果对象创建后，给对象的属性或方法进行修改或添加，那么等于给这个对象创建了一个自己的属性和方法，在删除时，只能删除对象被修改或者添加的成员
